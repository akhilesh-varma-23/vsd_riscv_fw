# VSDSquadron Mini Core

This repository is the official starter repository for the **VSDSquadron Mini Embedded Firmware Internship** by VLSI System Design (VSD).

The goal of this repository is to introduce participants to:
- Embedded firmware structure
- Library and API design
- How application code uses firmware libraries
- Good code organization practices used in industry

This repository will be extended throughout the internship.


## How to start

All participants must begin with **Task 1**.

Go to:
task1/README.md


and follow the instructions carefully.



## Important Notes

- This internship focuses on **embedded firmware development**
- Programming language: **C**
- Architecture: **RISC-V**
- No RTL / Verilog / HDL is used in this program
- Initial tasks do not require the physical board
